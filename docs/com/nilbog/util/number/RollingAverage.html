public function RollingAverage(count:uint = 10, startValue:Number = 0)

Constructor.

Parameters
count:uint (default = 10) — Number, the number of past and current values to recall when calculating this rolling average. Optional, defaulting to 10.
 
startValue:Number (default = 0) — Number, if provided, the rolling average is pre- populated so that it behaves as though the average was already updated count times with this value.
