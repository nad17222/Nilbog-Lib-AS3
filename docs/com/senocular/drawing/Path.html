Packagecom.senocular.drawing
Classpublic class Path
InheritancePath Object

The Path class saves a collection of line drawing commands to represent a path. Using the path class you can draw all of or a segment of that path in a movie clip or get locations and orientations along the path to have objects follow it. Note: People using Flash MX 2004 will have to use a custom flash.geom.Point class

public function circleTo(cx:Number, cy:Number, x:Number, y:Number):void

Creates a circle segment in the path from the current drawing position to (x, y) using a third intermediary point (cx, cy) which lies on the circular path between the current drawing position and (x, y). The current drawing position is then set to (x, y).

Parameters

cy:Number — An integer that specifies the vertical position of the control point of the circle segment
 
x:Number — An integer that specifies the horizontal position of the end anchor point of the circle segment
